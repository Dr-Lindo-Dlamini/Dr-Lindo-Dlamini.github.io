Getting Started with R

1. Writing and Running a Script

  • Open RStudio and click File > New File > R Script.
  • Write your code in the script.
  • Click Run or press Ctrl + Enter to execute.

2. Understanding RStudio Interface

  • Console: Where you type and execute commands.
  • Script Editor: Write and save multiple lines of code.
  • Environment: Stores variables and data.
  • Plots & Packages: Displays graphs and installed libraries.

3. Basic Commands

3.1 Arithmetic Operations

2 + 3   # Addition
5 - 2   # Subtraction
4 * 3   # Multiplication
10 / 2  # Division
2^3     # Exponentiation

3.2 Creating Variables

x <- 10   # Assigns 10 to x
y <- 5    # Assigns 5 to y
z <- x + y  # Adds x and y
z  # Prints result

3.3 Data Types

num <- 10.5    # Numeric
text <- "Hello"  # Character
bool <- TRUE   # Logical (TRUE or FALSE)

# Check the data type of each variable
class(num)   # Returns "numeric"
class(text)  # Returns "character"
class(bool)  # Returns "logical"

4. Working with Vectors

numbers_100 <- c(1:100)  # Create a vector with numbers from 1 to 100

numbers <- c(1, 2, 3, 4, 5)  # Create a vector
numbers[2]  # Access the second element
length(numbers)  # Check the length of the vector

5. Creating and Using Data Frames

data <- data.frame(Name = c("Alice", "Bob", "Thandeka","Londeka","John","Zama"), Age = c(25,13,40,31,17,6))
data  # View data frame
data$Age  # Access Age column

# To access Thandeka's age:
data$Age[data$Name == "Thandeka"]

6. Importing and Exporting Data

6.1 Importing Data

data <- read.csv("data.csv")  # Load CSV file
data  # View data

6.2 Exporting Data

write.csv(data, "output.csv")  # Save data to CSV

7. Basic Plotting

x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)
plot(x, y, type="o", col="blue", main="Simple Plot")  # Create a plot

8. Installing and Loading Packages

install.packages("ggplot2")  # Install package
library(ggplot2)  # Load package

9. Creating and Naming Objects in R

apples.sum <- 3 + 5
apples.sum  # View stored value

# Assign names to values:
John <- 3
Thandeka <- 5
apples.sum <- John + Thandeka
apples.sum

# Start names with a letter and use letters, numbers, _, and ..
my_value <- 10
long_variable_name <- 2.5

10. Updating and Using Objects

apples.sum <- apples.sum * 2
apples.sum  # Updated value

11. Adding Comments

# This is a comment
apples.sum <- 3 + 5  # Storing sum of apples

12. Using Functions

sqrt(4)  # Square root function
round(3.14159, digits = 2)  # Rounding function

13. Subsetting Vectors

numbers[2]       # Second element
numbers[c(1, 3)] # First and third elements

14. Summary

  • Use <- to assign values.
  • Use # for comments.
  • Use functions to perform tasks.
  • Use vectors and data frames to store data.
  • Keep naming conventions consistent.
